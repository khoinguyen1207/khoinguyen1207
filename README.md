Features:

### 🐤 Twitter Clone

A Twitter clone built with Node.js (Express.js)

**Tech:** Node.js, Express.js, TypeScript, MongoDB, Docker, AWS (S3, SES), JWT, OAuth, CI/CD with GitHub Actions

Features:

- Multi-tier authentication with JWT (access + refresh tokens) and Google OAuth
- Full social media features: tweets, comments, likes, retweets, follow/unfollow, bookmarks
- Advanced search with MongoDB text indexing and multiple filters
- Media handling: auto-resize images with Sharp, upload to AWS S3, HTTP range requests for video streaming
- Security: Helmet.js, CORS, rate limiting (100 req/min), SHA-256 password hashing
- Production deployment with Docker Compose, CI/CD
- Structured logging, environment validation, and a layered architecture pattern

**Tech:** Golang, Gin, PostgreSQL, Redis, RabbitMQ, JWT, Promtail, Loki, Grafana
